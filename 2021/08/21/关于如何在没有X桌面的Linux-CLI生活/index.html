关于如何在没有X桌面的Linux-CLI生活

前言

一直以来，各大主流Linux发行版(或者说主流桌面)，一直为个人用户桌面体验上下足了功夫，因为有着Windows和Mac这俩个桌面大户，要摆脱黑乎乎终端的偏见，吸引更多的新用户加入Linux这个大圈子，不得不说，现在Linux桌面的发展已经十分不错了(不错到臃肿的程度了)，没必要一味地追求到于上述俩一致的桌面体验。

二说，不大喜欢Xorg，可惜Wayland仍未成熟，也就勉强能用，实际开程序还是得要XWayland…主流DE也没几个支持,所以说如果要离开X桌面,该如何在CLI下生活呢

Step-1 :一个简洁的发行版

ArchLinux —- 追求实用简洁的KISS原则发行版

Gentoo Linux —- 高度自由化的元发行版

Linux From Scratch —- 从零选择一切的一本书

事实这里对发行版没有多大需求,只是因为上述的发行版相对于自带桌面的发行版整理起来会简单一些,都是命令行安装,需要有一定能力. LFS>Gentoo>Arch
